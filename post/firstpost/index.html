定期的に生エシャレットを過食する儀式の翌日。

お名前.comでドメイン取ってみてから早2ヶ月。
チマチマとブログ枠組みの妄想だけしてみていたのですが、
いい加減カタチだけでも、と思って作りました。

活用・参考させていただいたところ

目新しいモノは全然ない。

  • GitHub Pages
    無料でホスト。
  • お名前.com
    安い。常時セールな印象。
  • Hugo
    静的がいいよな、というのと、何か速そうなことを言っている人を多く見かけた。
  • Font Awesome
    右上のとしか使ってないけどな。
    ちょっとだけカッコつけたかっただけ。エシャレットっぽいアイコン探したけどなかった。
  • Angel’s Ladder
    Hugoのテーマ。本家のShowcasesが余りに微妙なので彷徨い歩いたところ、日本の方でシンプルに作られてるものを見つけた。 有難いかぎり。
  • 自分のGitHub Pagesと独自ドメインの設定方法
    ここだけちょっと悩んだ。今回ルートドメイン(tamanishi. net)をGitHub Pagesに繋げようと思ってた。
    んだけど、お名前.comのDNSだとルートドメインにはALIAS設定できないらしい、だとか
    DozensがルートドメインへのALIASをサポート(有料)してるらしい、とか
    Gehirnが2ドメインまで無料でルートドメインへのALIASをサポートしてたんだけど止めるらしい、とか
    DNSimpleは普通にサポート(有料)してるとか
    いろんな情報を見たんだけど、結局このQiitaの記事で事足りたという。

何にせよ、数年前に比べて簡単にキレイなものが作れるようになった。デプロイも何もかも簡単だ。 スバラシイ。
